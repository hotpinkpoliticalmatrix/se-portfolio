Now more than ever, consumers are holding corporations accountable for putting profits over people. However, the convenience of sites like Walmart & Amazon make it easier than ever to shop without much thought. AltCart is a chrome extension that provides users the opportunity to checkout their cart products from alternative, smaller retailers.
+ Built out the app's initial configuration framework, including its chrome manifest, webpack and babel configs, express middleware
+ Developed highly scalable front-end architecture using Redux state management
+ Implemented SerpAPI's remote google search library for fast & up-to-date data rendering to provide alternative retailers to users
